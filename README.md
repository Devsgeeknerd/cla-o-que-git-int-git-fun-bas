# O Que é Git?

***Conteúdo da Aula:***

Basicamente, o Git é um sistema para controle de versão de arquivos muito utilizado por desenvolvedores para controlar as modificações de seus projetos e arquivos.

Com o Git, podemos criar todo o histórico de alterações dos arquivos de código-fonte de projetos e, assim, facilitar o gerenciamento dos nossos projetos.

Além disso, o Git facilita a colaboração de diversos desenvolvedores trabalhando em um único projeto, já que este possui ferramentas para análise e redução de conflitos quando um mesmo arquivo é editado por mais de um desenvolvedor.

Sendo assim, durante todo o curso, veremos alguns recursos fundamentais que todo desenvolvedor precisa conhecer para trabalhar com o Git.
